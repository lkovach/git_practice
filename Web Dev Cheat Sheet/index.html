Web Dev Cheat Sheet


Video/Audio/Embed Tags

Tag Description Example
<video> Adds video to a web page. Video is not a self-closing element. <video src="video.mp4"></video>
<audio> Adds audio to a webpage. Like video, this is not a self-closing element <audio src="some audio.mp4"></audio>
<embed> Can be used for any media type and is self-closing <embed src="somemedia.xxx">


Video Attributes

Attribute Description
src Identifies the source of the video. If the source is unavailable, an error message should be displayed to the user.
controls Shows the standard video controls: play/pause, fullscreen, and audio controls
autoplay Starts the video automatically when the page is loaded
loop Plays the video continuously


Audio Attributes

Attribute Description
src Identifies the source of the audio. If the audio is not available, an error message should be displayed to the user.
controls Provides play/pause and volume controls
autoplay Starts the audio automatically when the page loads


Header, Footer, and Nav

Tag Description Example
<header> A container usually for either navigation links or introductory content containing h1 to h6 headings. <header>...</header>
<footer> Contains information such as: contact info, copyright info, Terms of Use, Site Map, Reference to top of page links <footer><p>...</p></footer>
<nav> Used to define a block of navigation links such as menus and tables of contents <nav>*bunch of links here*</nav>
